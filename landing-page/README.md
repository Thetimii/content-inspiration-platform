# Lazy Trends Landing Page

A modern, animated landing page for Lazy Trends - a service that helps content creators discover viral trends and create engaging content effortlessly.

## Features

- Clean, modern design with subtle animations
- Fully responsive layout
- Interactive elements with smooth transitions
- Optimized for performance and SEO

## File Structure

```
landing-page/
├── index.html          # Main HTML file
├── styles.css          # CSS styles
├── script.js           # JavaScript for interactive elements
├── images/             # Image assets
│   ├── lazy-trends-logo.svg
│   ├── icon-signup.svg
│   ├── icon-analysis.svg
│   ├── icon-create.svg
│   └── ...
└── README.md           # This file
```

## Usage

1. Simply open `index.html` in a web browser to view the landing page.
2. To deploy, upload all files to your web server.

## Customization

- **Colors**: Edit the CSS variables at the top of `styles.css` to change the color scheme.
- **Content**: Modify the text in `index.html` to match your specific messaging.
- **Images**: Replace the placeholder images in the `images/` directory with your own.

## Browser Support

This landing page is compatible with all modern browsers:
- Chrome
- Firefox
- Safari
- Edge
